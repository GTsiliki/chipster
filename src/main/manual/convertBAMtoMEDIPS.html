Convert BAM files to MEDIPS input files

Description

Convert one or more BAM files to a format suitable for input to MEDIPS analysis tools.

Parameters

None

Details

None

Output

As many files as there were original BAM files. The files are names as MEDIPS-inputX.tsv, where X is a running number. BAM files are read in alphabetical order, and the MEDIPS files are written in the same order.
