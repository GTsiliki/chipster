Utilities / Filter by column value

Description

Filter genes on the basis of values in one of the data columns.

Parameters

  • column (...) [empty]
  • cutoff (-100000...100000) [1]
  • smaller.or.larger (equal-to, smaller-than, larger-than, within, outside) [smaller-than]

Details

Filters rows in a table based on continuous values in the selected data column (for example after using the tool "Calculate descriptive statistics"). When filtering for discrete matches you have to set the type parameter to "discrete", enter the filtering term to the "discrete match" field, and set the last parameter to equal-to. The options "within" and "outside" can be used to filter based on a symmetrical range of values, applicable for instance when filtering for "at least 2-fold up- and downregulated genes".

Output

A tabular text file containing only the genes that pass the filter.
