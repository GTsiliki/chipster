Pathways / Hypergeometric test for cytobands

Description

Finds chromosomal locations, i.e., cytobands for which the genes in the selected dataset are either over- or under-represented.

Parameters

  • P value threshold (0-1) [0.05]
  • Search for over- or under-represented genes (under, over) [over]

Details

This tool takes a filtered data set and goes through all cytobands. In order to avoid the identification of directly related regions with considerable overlap of genes, this tool uses a method which conditions on all subregions that are themselves significant at a specified p- value cutoff. First the subregions are tested and the genes of the significant subregions are removed before testing the larger regions.

Output

Output is an HTML-file with a list of significant cytobands.
