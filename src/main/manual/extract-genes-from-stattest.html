Utilities / Extract genes using a p-value

Description

Saves genes from a statistical test result as a new data set.

Parameters

  • P-value threshold (0-1) [0.05]
  • P-value column

Details

If run on a data set that contains p-values from a statistical test, extracts all the genes that have a p-value equal to or lower than the specified threshold. These genes are saved as a new data set that can be used for further analysis.

Output

A tab-delimited text file.
