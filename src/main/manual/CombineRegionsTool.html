Matching genomic regions / Combine region files

Description

Given two BED files, this tool returns combined regions from both files. Also known as union.

Parameters

  • Merge overlapping regions (yes, no) [yes]
  • Minimum number of overlapping bases, if merging (1-1000000000) [1]

Details

Output

Output is a BED file.

References
