Analysis tools for next generation sequencing data

Every tool is described in detail on its own help page. The help page is divided into four parts: General description, Parameters, Details on how the tool works, and Output where the results are described. The Parameters part contains the list of parameters, their possible values (1-100) and their default values [10].
