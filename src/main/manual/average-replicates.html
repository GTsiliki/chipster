Utilities / Average replicates

Description

Calculates an arithmetic average of the samples labeled as belonging to the same group.

Parameters

  • Group

Details

User needs to choose from phenodata one column that specifies which samples need to be averaged together. Those belonging to the first group, need to be labeled as "1", those belonging to the second group as "2", and so forth. Samples belonging to the first group are then averaged together, as are the samples in the second group.

Output

A table containing averages of the groups.
