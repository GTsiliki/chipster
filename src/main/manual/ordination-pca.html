Statistics / PCA

Description

Calculates principal components for the selected gene list, and saves them for further use.

Parameters

  • Data to use for the analysis (genes, chips) [genes]
  • Variability to explain (0...100) [80]

Details

A principal component analysis is run for either genes or the samples. Principal components are retained until the explained variability exceeds the specified value.

When PCA is run on samples, it's output can be visualized in Chipster GUI using the 3D scatterplot.

Output

A gene list with principal components (those are not expression values).
