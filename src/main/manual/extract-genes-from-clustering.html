Utilities / Extract genes from clustering

Description

Saves genes in a specific cluster as a new data set.

Parameters

  • Cluster number (1-1000) [1]

Details

If run on a result from K-means, SOM or QT-clustering extracts the genes in a specified cluster, and saves the data for these genes as a new data set. This data set can then be used for further analyses.

Output

A tab-delited text file.
