Pathways / Boolean network

Description

Infers a Boolean network from a timeseries data.

Parameters

  • Number of inputs per function (1...100) [2]
  • Number of genes (1...100) [50]
  • Discretization method (median, cutoff) [median]

Details

Only time series dataset where there is one replicate per time point can be analyzed. Genes that are periodically expressed are first selected, and 50 top genes are sent to bestim-program for further analysis. Bestim infers the best Boolean network for the genes. The best network is then returned to the user.

Output

A text file containing the best Boolean network.
