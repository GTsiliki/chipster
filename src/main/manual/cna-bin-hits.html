CNA-seq / Count reads in fixed bins

Description

Count reads in non-overlapping bins of fixed size.

Parameters

  • organism (human) [human]
  • human genome build (GRCh37/hg19) [GRCh37/hg19]
  • bin size (1 kbp, 5 kbp, 10 kbp, 15 kbp, 30 kbp, 50 kbp, 100 kbp) [30 kbp]

Details

The genome is divided into non-overlapping bins of the specified size, and the number of reads counted for each bin. The read counts are also adjusted for GC content.

Output

Original and GC corrected read counts per bin.
