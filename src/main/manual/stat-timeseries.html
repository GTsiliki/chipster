Statistics / Time series

Description

Finds periodically expressed genes from the selected gene list.

Parameters

  • Analysis type (periodicity, ica) [periodicity]
  • P-value cut-off (0...1) [0.05]
  • Apply Benjamini-Hochberg FDR correction to p-values (yes, no) [yes]
  • Standard deviation for ica (0...10) [2]
  • Image width (200...3200) [600]
  • Image height (200...3200) [600]

Details

Periodically expressed genes can be found using either the Fisher's G test or independent component analysis (ICA). For the Fisher's test, genes that have the corrected p-value less than the specified threshold are returned. For ICA, genes that are within the specified standard deviation from the mean, are returned.

Output

A list of genes expression and p-values (periodicity analysis) or a list of genes with expression values and their cluster membership (ICA). For ICA, a plot of genes in the clusters is also generated.
