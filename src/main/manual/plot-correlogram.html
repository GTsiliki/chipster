Visualisation / Correlogram

Description

Creates a correlogram for a normalized dataset.

Parameters

  • Image width (200-3200) [600]
  • Image height (200-3200) [600]

Details

Correlogram visualizes the pairwise correlations of the samples. The upper triangle of the plot consists of pairwise scatter plots. The lower triangle visualizes bot the direction of the correlation using thin lines inside the colored boxes, and the magniture of correlation using the colors. The darker the color, the higher the correlation.

Output

A PNG image containing the correlogram:
