BEDTools mergeBed

Description

Merges overlapping BED/GFF/VCF entries into a single interval.

Parameters

  • Force strandedness (yes, no) [no]
  • Report the number of BED entries that were merged (yes, no) [no]
  • Maximum distance between features allowed (Integer) [0]
  • Report the names of the merged features separated by semicolons (yes, no) [no]
  • Report the scores of the merged features (yes, no) [no]
  • Score report type (sum, min, max, mean, median, mode, antimode, collapse) [sum]

Details

mergeBed combines overlapping or “book-ended” (that is, one base pair away) features in a feature file into a single feature which spans all of the combined features.

Output

Output is a BED file.
