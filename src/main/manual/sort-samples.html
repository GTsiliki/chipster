Utilities / Sort samples

Description

Order the samples in the data using a selected phenodata column.

Parameters

  • Column (...)

Details

Specify a phenodata column that should be used for ordering the samples. This column can contain any data numerical values, but the samples are ordered in the ascending order. However, several samples can contain the same number, which amounts to ordering the samples in groups.

Output

A modified dataset and new phenodata file.
