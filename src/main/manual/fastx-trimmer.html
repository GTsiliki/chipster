Utilities / Trim reads

Description

Given a FASTQ file, this tool trims reads as specified by the user.

Parameters

  • First base to keep (1-100) [1]
  • Last base to keep (1-100) [50]

Details

You can use this tool for example for trimming low quality ends of your reads. If quality control tools have indicated that only some reads have low quality ends, you might prefer to use the tool "Filtering / Filter out low quality reads" which will take the base quality values into account.

Output

A FASTQ file containing the trimmed reads.

Reference

This tool is based on the FASTA/Q Trimmer tool of the FASTX package.
