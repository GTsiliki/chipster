Preprocessing / Filter by column term

Description

Filter genomic regions on the basis of information in one of the data columns.

Parameters

  • column (...) [empty]
  • match.term (...) [empty]

Details

Filters rows in a table containing genomic regions based on textual terms in the selected data column. You can filter for text matches such as "upstream" or "downstream" in the output file from the "Find nearest genes for regions" tool. Notice that rows where the match term is only a part of the whole term are also passing the filter.

Output

A tabular text file containing only the set of genomic regions that pass the filtering criteria.
