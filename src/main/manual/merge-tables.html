Utilities / Merge tables

Description

Combines two tables using row names.

Parameters

None

Details

None

Output

A tab-delimited text file containing all the rows that can be combined from the two tables. This file can be used for further analyses, such as statistical tests.

The result file contains all the fields from both tables, so before using it for further analyses, you might want to delete duplicates fields, such as data columns, using the tool Delet columns in the Utilities category.
