Normalize to specific genes

Description

This tool allows the user to normalize data to specific (positive) control genes.

Parameters

none

Details

An average of the specified genes on each chip is calculated and subtracted from all the values on the same chip. This tool requires two input files: the normalized data and a list of gene identifiers. The identifier list must have a title row with text "identifier" and contain gene identifiers (row names) used in the data file.

Output

A new normalized dataset.
