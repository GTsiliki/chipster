Preprocessing / Remove missing values

Description

Removes all genes that have at least one missing value in the experiment.

Parameters

  • None

Details

All genes that contain at least one missing value are excluded from the data. This tool is suggested to be used after cDNA/Agilent normalization, since these might generate missing values that need to be processed before, e.g., clustering or statistical tests. A good preprocessing order might be to first impute these missing values, and after that remove all the remaining missing values using this tool.

Output

A tab-delimited text-file containing the cut-down dataset.
