Utilities / Statistics for BAM

Description

Given a BAM file and an index file (.bai) for it, this tool counts how many alignments there are, for example per each chromosome.

Parameters

None

Details

Please note that in addition to BAM file you have to provide an index file. You can create the index file using the tool "Index BAM".

Output

Output is a table (bam-stats.tsv) listing how many reads align to the different chromosomes and what the chromosome lengths.

References

This tool is based on the SAMtools package.
