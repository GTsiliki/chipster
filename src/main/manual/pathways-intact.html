Pathways / Protein interactions from IntAct

Description

Maps genes to protein interactions in the IntAct database. This service is provided by the EBI.

Parameters

  • Input type (affyid, uniprot) [affyid]

Details

This tool takes a filtered data set (e.g. a list of differentially expressed genes) and maps the genes to IntAct protein interactions. It works only with Affymetrix and UniProt identifiers.

Output

Output is a list of interactions as a text file (allows sorting by columns) and as an html file (with hyperlinks to the corresponding entries in the IntAct database and UniProt database).

References

More information can be found at the IntAct website and in the following article:

Kerrien et al: Open source resource for molecular interaction data. Nuc Acids Res 35:D561-D565 (2007)
