Utilities / Extract genes from GO term

Description

This tool extracts the probes for genes that map to a user specified gene ontology term.

Parameters

  • The gene ontology ID of the term to match.

Details

Given a gene ontology identifier, for example GO:0051090, the tool identifies all the probes that target genes in the input data that match to that term query.

Output

The portion of the input table containing only the rows for probes that were succesfully mapped to the gene ontology term.
