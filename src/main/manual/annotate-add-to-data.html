Annotation / Add annotations to the data

Description

This tool appends annotations to the expression data for the selected Affymetrix, Illumina or Agilent gene list.

Parameters

None

Details

Currently, Affymetrix probe set IDs, Illumina Target IDs and Agilent probe names can be annotated.

Output

This tool writes a data file where annotation columns are appended to the gene expression value columns. Annotations contain chromosome location, pathway information, and links to databases such as Pubmed and RefSeq.

References

This tool utilizes the Bioconductor package annaffy.
