Filter / Flags

Description

Filter genes on the basis of flag values.

Parameters

  • Flag to filter by (P, A, M, A.or.M, M.or.P) [P]
  • Number of chips (1...10000) [2]

Details

Selects the genes for which the flag is equal to the selected condition in at least the specified number of arrays. The "A.or.M" and "M.or.P" options allow the inclusion of two flag conditions in one single filtering step.

Output

A tabular text file containing only the genes that pass the filter.
