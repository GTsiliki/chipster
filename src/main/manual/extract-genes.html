Utilities / Extract genes

Description

Extracts the specified number of genes from the top of an expression data table.

Parameters

  • Number of genes (1-1000000) [50]

Details

This tool is typically used in conjunction with the Sort genes -tool. For example, you could run the Calculate descriptive statistics tool, sort genes according to the standard deviation column, and extract the 50 top-most genes.

Output

A tab-delimited text file containing the specified number of rows. This file can be used for further analyses, such as statistical tests.
