Statistics / Estimate sample size

Description

Calculates various sample size related estimates based on normalized data set.

Parameters

  • Effect size, i.e., fold change (0...100) [2]
  • Alpha, i.e., p-value (0...1) [0.05]
  • Beta, i.e., power (0...1) [0.05]
  • Group size, i.e., sample size

Details

Calculates several estimates related to sample size and power based on a normalized dataset. These estimates are:

  • Sample size to detect at least an effect of the specified magnitude.
  • Power to detect at least an effect of the specified magnitude with the specified sample size.
  • Fold change that can be detected at certain power assuming the specified sample size.

Every estimate produces a separate image that can be visualized using the Chipster GUI.

Output

Three images visualizing the estimates produced by the tool.

References

This tools uses Bioconductor package ssize. Please site the following article:

Warnes, G. R., Liu, P. (2005) Sample Size Estimation for Microarray Experiments, submitted to Biometrics.
