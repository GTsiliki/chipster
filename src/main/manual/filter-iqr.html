Filter / Interquartile range

Description

Filter genes on the basis of their interquartile range.

Parameters

  • Interquartile range (0...10) [0.5]

Details

Genes are filtered on the basis of their interquartile range. The genes that have larger than the specified interquartile range are retained.

At least two chips are needed in order to be able to use this tool.

Output

A tabular text file containing only the genes that pass the filter.
