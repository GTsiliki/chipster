aCGH / Plot copy-number-induced gene expression

Description

Plot the expression levels of individual genes for a copy number vs. expression comparison. This tool must be run on the output from the tool Test for copy number induced expression changes - cn-induced-expression.tsv.

Parameters

  • Gene IDs of the genes to be plotted (...) [1]
  • Image width (200...3200) [600]
  • Image height (200...3200) [600]

Details

For each gene, the arrays are divided into two groups either "loss vs. no-loss (normals and gains)" or "no-gain (losses and normals) vs. gain" depending on the amount of gains and losses for that particular gene. This tool plots the expression values of specified gene(s) across the two groups. Each array is plotted as a circle, and the center of the circle marks the expression level of the particular gene for that array. The diameter of the circle tells the probability of the particular copy number call (e.g. a large circle in the "loss" group means that for this particular array, a call for loss was made with high probability). Group averages are shown in red.

Output

An image showing the expression levels for the gene(s) of interest in the two groups defined by copy number.

References

Wieringen et al. (2009) Nonparametric testing for DNA copy number induced differential mRNA gene expression. Biometrics 65: 19-29
