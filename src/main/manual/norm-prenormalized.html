Normalization / Process prenormalized

Description

If you bring in data which has been normalized by some other software, this tool converts the data to Chipster format (by adding the text "chip." in front of the expression value columns) and generates the phenodata file.

Parameters

  • Chiptype (empty) [empty]

Details

The normalized data file needs to be imported to Chipster using the Import tool. Every column containing normalized expression values need to be specified as Sample in the Import tool.

If you want to be able to use analysis tools requiring annotation information, you need to specify the chiptype using the chip's annotation package name (they are listed in brackets after the chip name here)

Output

A new normalized dataset and a phenodata file.
