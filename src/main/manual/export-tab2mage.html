Utilities / Export tab2mage

Description

This tools exports normalized data in a format suitable for batch submission to ArrayExpress database. Only obligatory fields are output. An empty file is output, and user must fill it in before submitting it to ArrayExpress.

Parameters

  • None

Details

See http://tab2mage.sourceforge.net/docs/index.html for more information about the tab2mage format.

Output

A formatted text file suitable for a direct batch submission to ArrayExpress database.
