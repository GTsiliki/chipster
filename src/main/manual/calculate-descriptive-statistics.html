Statistics / Calculate discriptive statistics

Description

This tool calculates descriptive statistics to all genes or samples in the selected data set.

Parameters

Calculate descriptives for (genes, chips) [genes]

Details

Calculates an arithmetic average, median, standard deviation, coefficient of variation, standardized expression, minimum, maximum, range, and inter-quartile range for all genes or chips.

Output

Returns two tab delimited text files. Descr-stats.tsv is the original datafile with added columns that contain the descriptive statistics. This files can be further filtered e.g. by using the filter-by-column tool. Descriptives.tsv contains the descriptives but no expression values. This file can be used for drawing histograms and box plots of the descriptives.
