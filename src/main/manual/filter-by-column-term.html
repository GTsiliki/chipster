Preprocessing / Filter using a column term

Description

Filter genes on the basis of information in one of the data columns.

Parameters

  • column (...) [empty]
  • match.term (...) [empty]
  • exact.match (yes, no) [yes]

Details

Filters rows in a table based on textual terms in the selected data column. You can filter for text matches such as GO terms (after using the tool "Add annotations to data" tool). Notice that in order to identify rows in a data table where the match term is only a part of the whole term the "exact.match" parameter needs to be set to "no".

Output

A tabular text file containing only the genes that pass the filter.
