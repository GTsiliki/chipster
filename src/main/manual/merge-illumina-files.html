Utilities / Merge Illumina files

Description

Combines two Illumina BeadSymmary data tables that have been imported to the system. This is especially useful, if the experiment is divided over a number of BeadSummary data files. After combination a single BeadSummary data file is generated. This file can then be normalized using the Illumina normalization tool.

Parameters

None

Details

None

Output

A BeadSummary data file.
