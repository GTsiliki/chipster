Utilities / Sort genes

Description

Sort genes according to the selected column in ascending or descending order.

Parameters

  • Column (empty, ...) [empty]
  • Method (ascending, descending) [ascending]

Details

Combining this tool with the Calculate descriptive statistics -tool and the Extract genes -tool allows you for example to sort the data according to the standard deviation and extract the 50 top-most genes.

Output

A tab-delimited text file sorted by values in the column specified by the user. This file can be used for further analyses, such as statistical tests.
