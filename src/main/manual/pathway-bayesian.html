Pathways / Bayesian network

Description

Infers a Bayesian network from a timeseries data.

Parameters

  • P-value threshold (0...1) [0.05]
  • Image width [600]
  • Image height [600]
  • Coloring scheme (solid, expression) [solid]

Details

Only time series dataset where there is one replicate per time point can be analyzed. First, partial correlation between genes are calculated, significance tests for the correlations are calculated and only the genes that are significant at the selected p-value level are included in the bayesian network that is inferred at the last phase of the analysis.

Output

A colored image visualizing the pathway.
