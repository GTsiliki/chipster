ChIP-seq / Find unique and annotated genes

Description

This tool takes a list of ENSEMBL gene identifiers, removes duplicates and fetches annotation information. The output file is compatible with downstream tools used for example for pathway analysis.

Parameters

  • Genome (Human hg18, Mouse mm9, Rat rn4) [none]

Details

In order to fetch annotation information the tool uses ENSEMBL gene identifiers as input, which it converts to entrez gene id:s through the biomaRt interface to ENSEMBL. The output file is compatible with downstream tools used for example for pathway analysis.


Output

The analysis output is a table of annotated genes, which is compatible with downstream tools used for example for pathway analysis.


References

The tool makes use of the following Bioconductor packages:

  • biomaRt

  • ChIPpeakAnno
