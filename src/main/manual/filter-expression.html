Filter / Expression

Description

Select genes on the basis of expression.

Parameters

  • Cut-off for over-expressed genes (-10000...10000) [100]
  • Cut-off for under-expressed genes (-10000...10000) [5]
  • Number of chips (1...10000) [1]
  • Scale to the same mean (yes, no) [no]
  • Included genes (outside-the-range, inside-the-range) [inside-the-range]

Details

Cut-offs for over- and under-expressed genes can be selected. The user can also choose the number of arrays on which the expression needs to fulfill the selected cut-offs. By default, the arrays are not scaled to their mean values, which is most appropriate for 1-color arrays, and the genes that fall inside the specified expression range are selected, which makes sense when trying to filter low quality or saturated probes. It is also possible to select the genes that fall outside the selected range, which, when combined with the mean scaling option, is a convenient way to filter out invariant genes in 2-color experiments.

Output

A tabular text file containing only the genes that pass the filter.
