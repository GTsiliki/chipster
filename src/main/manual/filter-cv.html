Filter / Coefficient of variation

Description

Filter genes on the basis of their coefficient of variation (standard deviation / mean).

Parameters

  • None

Details

Calculates a coefficient of variation (CV) for every gene, and returns the genes where CV is bigger than the median of all calculated CVs. In other words, the genes that have more similar values for all observations are are filtered away.

Output

A tabular text file containing only the genes that pass the filter.
