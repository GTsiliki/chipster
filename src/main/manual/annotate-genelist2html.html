Annotation / Affymetrix, Illumina or Agilent gene list

Description

This tool creates an HTML page containing annotations for a selected Affymetrix, Illumina or Agilent gene list.

Parameters

  • Column containing expression values [empty]
  • Column containing p-values [empty]

Details

Currently, Affymetrix probe set IDs, Illumina Target IDs and Agilent probe names can be annotated.

If the annotated dataset contains p-values or expression values, those can be included in the annotation table. P-values and expression values can be calculated using some statistical test, especially empirical Bayes, or using the utility toll for calculation fold changes between exactly two groups.

Output

Tool writes an HTML and a text file with annotations for the genes. This annotation page contains information on the probe location in the chromosomes, links to outside databases, such as, Pubmed and RefSeq, and GO and KEGG pathway information.

References

This tool utilizes the Bioconductor package annaffy.
