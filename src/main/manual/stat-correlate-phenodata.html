Statistics / Correlate with phenodata

Description

Finds genes that have a high correlation with the selected phenodata variable.

Parameters

  • Correlation threshold (0-1) [0.95]
  • Correlation method (pearson, spearman) [pearson]
  • Phenodata column

Details

This tools goes through all the genes in the data, calculates their correlation coefficients against the selected phenodata variable, and saves those that exceed the selected correlation threshold.

Output

A list of genes that have a correlation coefficient higher or equal to the one selected.
