Statistics / Association analysis

Description

Checks for deviations from Hardy-Weinberg equilibrium or for differences in genotype frequences between two groups using SNP genotype data.

Parameters

  • Test for (Hardy-Weinberg, association) [association]
  • Phenodata column (...)

Details

The test for Hardy-Weinberg equilibrium is run using the samples from the control only.

The association test runs the analysis using both recessive and dominant models.

Output

The genotype data attached with the p-value from the selected analyses.
