Utilities / Intersect lists

Description

Identifies the intersection, or union, between two or three data tables that share one or more columns with common identifiers.

Parameters

  • common.column (...) [empty]
  • operation (intersection, union) [intersection]
  • Image width (200-3200) [600]
  • Image height (200-3200) [600]

Details

This tool takes two or three tables of data that share at least one column with common identifiers, identifies the common ones (intersection) or the ones that appear in any of the tables (union) and returns a table with these. In addition, a table summarizing all the possible combinations of intersections, unions and unique instances as well as a Venn diagram for a visual interpretation of this are also returned.

Output

Output consist of two result tables and one Venn diagram plot.
