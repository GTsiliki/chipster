Utilities / Extract genes from KEGG pathway

Description

This tool extracts the probes for genes that map to a user specified KEGG pathway.

Parameters

  • The ID of the KEGG pathway.

Details

Given a KEGG pathway identifier, for example 04910, the tool identifies all the probes in the input data that target genes that map to that pathway.

Output

The portion of the input table containing only the rows for probes that were succesfully mapped to genes in the query pathway.
