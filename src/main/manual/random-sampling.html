Utilities / Random sampling

Description

Create a random subset of the data, either with respect to genes or samples, of a given size.

Parameters

  • What to sample (genes, chips) [genes]
  • Sample size (Any decimal number from 0 to 100 [10]

Details

Generates a randomly selected subset of the data (either with regard to genes or samples), which makes up a user-specified percentage of the input data.

Output

A tab-delimited text file with intensity values suitable for further analyses, such as statistical tests.
