Utilities / Search by correlation

Description

Search similarly expressed gene using correlation.

Parameters

  • Correlation method (pearson, spearman) [pearson]
  • Correlation cutoff (-1...1) [0.95]

Details

Expects a name of gene. Name can be, for example, Affymetrix probeset ID. Genes that are similarly expressed are found.

Output

A tab-delimited text file of the similarly expressed genes (including the one used in the search). This file can be used for further analyses, such as statistical tests.
