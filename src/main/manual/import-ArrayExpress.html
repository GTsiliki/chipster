Utilities / Import from ArrayExpress

Description

Imports Affymetrix raw data directly from ArrayExpress, and normalizes it using the RMA algorithm.

Parameters

  • Accession number of the experiment

Details

Loads the specified dataset from ArrayExpress to Chipster, normalizes it using the RMA algorithm, calculates detection calls using the MAS5 algorithm and creates a phenodata for it.

Output

Normalized dataset, including detection calls, and a corresponding phenodata table.
