Pathways / Associations to Reactome pathways

Description

Maps genes to pathways in the Reactome database. This service is provided by the EBI.

Parameters

  • Input type (affyid, uniprot) [affyid]

Details

This tool takes a filtered data set (e.g. a list of differentially expressed genes) and maps the genes to Reactome pathways. It works only with Affymetrix and UniProt identifiers.

Output

Output is a list of pathways as a text file (allows sorting by columns) and as an html file (with hyperlinks to the corresponding entries in the Reactome database).

References

More information can be found at the Reactome website and in the following article:

Matthews et al: Reactome knowledgebase of human biological pathways and processes. Nuc Acids Res 37:D619-D622 (2009)
