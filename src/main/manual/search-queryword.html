Utilities / Search by gene name

Description

Search for genes using the row names of data, gene symbols (HUGO names), or chromosome locations.

Parameters

  • search.for (rowname, symbol, chromosome.location) [rowname]
  • query (free text) [empty]
  • exact (yes, no) [yes]
  • mode (include, exclude) [include]

Details

Searches for the genes that have a description matching the query. Can be used, for example, for retrieving all genes in one chromosome. The "exact" parameter controls whether the query term should match exactly or be a part of the match, whereas the "mode" parameter allows the choice to include or exclude the matching genes from the output.

Output

Returns a tab-delimited text file containing the found genes. This file can be used for further analyses, such as statistical tests.
