CNA-seq / Plot copy number profile

Description

Plot copy number profiles of individual arrays from called CNA-seq data.

Parameters

  • samples to be plotted (1...n) [1]
  • chromosomes to be plotted (0...25) [0]
  • resolution (0..1) [0.1]
  • phenodata column with ploidy of samples

Details

Original ratios are shown in black, copy number calls in black, blue, or red for the ploidy copy number, gains, and losses, respectively.

Output

Copy number profile of individual CNA-seq sample(s).
