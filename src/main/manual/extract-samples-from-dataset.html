Utilities / Extract samples from dataset

Description

Creates a subset of the current data set.

Parameters

  • Group

Details

User needs to select from phenodata a column that contains the grouping of the data. This column needs to define exactly two groups that are coded with 0 and 1. The samples that are needed to create the new data set should be coded with 1s. The samples to be excluded from the new dataset are coded with 0s.

Output

A new data set with only the extracted sample in the tab-delimited text file, and a cropped phenodata that corresponds to the extracted data set.
