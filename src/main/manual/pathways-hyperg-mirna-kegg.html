Pathways / KEGG enrichment for miRNA targets

Description

Performs a statistical test for enrichment of KEGG pathways in the predicted gene targets of a list of miRNAs.

Parameters

  • P-value threshold (0-1) [0.05]
  • P-value adjustment method (none, BH, BY) [BH]
  • Minimum population (1-1000000) [10]
  • Species (human, mouse, rat) [human]

Details

This tool takes normalized data with miRNA IDs, fetches the target genes from miRBase, and runs a hypergeometric test for over-representation for KEGG pathways in this list. Analysis can be restricted to KEGG pathways which have a minimum, user-specified number of genes mapped to them (by default ten). The user needs to specify species for the miRNA data.

Output

Output is a text file with a list of significant categories.
