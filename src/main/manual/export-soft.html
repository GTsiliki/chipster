Utilities / Export SOFT

Description

This tools exports normalized data in a format suitable for batch submission to GEO database. Only obligatory fields are output. User can define whether to output an empty table that should be filled by the user or to output a prefilled table.

Parameters

  • None

Details

See http://www.ncbi.nlm.nih.gov/projects/geo/info/submission.html for more information about batch submmissions to GEO database.

Output

A formatted text file suitable for a direct batch submission to GEO database.
